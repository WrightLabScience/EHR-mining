This repository will contain R scripts, text files, and other data relevant to mining, cleaning, and analyzing electronic health record (EHR) data in the AMB_ETL schema as provided by R3.

Tasks will include:

* Cleaning organism name field in AST data
* Cleaning antibiotic name field in AST data
* Imputing missing AST results
* Cleaning drug name in medication dispense data (and hopefully medication administration data soon)
